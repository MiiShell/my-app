# Privacy Policy (Draft)

This app collects limited data strictly for providing utility features and advertising in a compliant manner.

- Advertising: Google AdMob (via google_mobile_ads). User Messaging Platform (UMP) used for GDPR consent in EEA.
- File access: Access to media/files only when you use File Finder/Cleanup. No automatic background deletion.
- Battery history (opt-in): Stores battery metrics locally using Isar. You can delete all data in Settings.
- Usage insights (opt-in): Requires Usage Access permission. Used only to display usage statistics to you.

You can withdraw consent or disable features any time in Settings. For questions, contact: privacy@yourdomain.com.

This is the markdown source for the hosted HTML at https://yourdomain.com/privacy-policy.html.
